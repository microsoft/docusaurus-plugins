## Usage

All code snippets for the given languages in the options will be extract by default.

    ```js
    // this code snippet will be extract to file
    let x = 'hello'
    ```

You will find the code below in the `snippets` folder.

```js
// this code snippet will be extract to file
let x = 'hello'
```


### `no-extract`

Add the `no-extract` meta on a snippet to exclude it from extraction.

    ```js no-extract
    // this code snippet will NOT be extracted
    ````
