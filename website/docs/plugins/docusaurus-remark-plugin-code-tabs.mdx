# ðŸ“¦ remark-plugin-code-tabs

Groups sequences of code section into a single tabbed element
using [Docusaurus tabs](https://docusaurus.io/docs/markdown-features/tabs).

## Usage

Read the [Code Tabs documentation](/docs/markdown-features/code-tabs).

## Configuration

Install the plugin

```bash npm2yarn
npm install @microsoft/docusaurus-remark-code-tabs
```

and add to your `docusaurus.configuration.js` file as other remark plugins.

```js
const config = {
    plugins: [
        ...,
        // highlight-start
        "@microsoft/docusaurus-remark-code-tabs"
        // highlight-end
    ]
}
```

If you are using `codesandbox`, install
the [codesandbox-button theme](./docusaurus-theme-codesandbox-button).
