# Remark plugin compile code

Execute tools, typically compiler, during build time
and inline the resulting output in the documentation.


````markdown
```z3wasm
(declare-const a Int)
(declare-fun f (Int Bool) Int)
(assert (= a 10))
(assert (= (f a true) 100))
(check-sat)
```
````
