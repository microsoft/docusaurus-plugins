## Rise4fun Extras

Rise4fun adds new Markdown features to help with writing documentation.

-   [Math Equations](./markdown-features/math-equations) are enabled by default
-   [Diagrams](./markdown-features/diagrams) are enabled by default
-   [CodeSandbox Button](./markdown-features/codesandbox-button) creates a CodeSandbox project on the fly
-   [Code Tabs](./markdown-features/code-tabs) show code sections in a tabbed view, with optional CodeSandbox button
-   [Compile Code](./markdown-features/compile-code) precompile snippets and displays output
-   [Side Editor](./markdown-features/side-editor) loads a snippet in a live editor in a split panel
-   [Code Element](./markdown-features/code-element) maps a fenced code section to a custom MDX theme element
