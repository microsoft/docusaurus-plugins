## Extras

Rise4fun adds new Markdown features to help with writing documentation.

-   [Math Equations](./math-equations) are enabled by default
-   [Diagrams](./diagrams) are enabled by default
-   [CodeSandbox Button](./codesandbox-button) creates a CodeSandbox project on the fly
-   [Code Tabs](./code-tabs) show code sections in a tabbed view, with optional CodeSandbox button
-   [Compile Code](./compile-code) precompile snippets and displays output
-   [Side Editor](./side-editor) loads a snippet in a live editor in a split panel
